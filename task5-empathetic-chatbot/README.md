# Task 5: Mental Health Support Chatbot

## Task Objective
The goal of this task is to build a supportive conversational agent by fine-tuning a small Large Language Model to provide empathetic responses for stress, anxiety, and emotional wellness.

## Dataset Used
* **Name:** EmpatheticDialogues
* **Source:** Facebook AI (via Hugging Face Datasets)

## Models and Techniques Applied
* **Base Models:** DistilGPT2 / GPT-Neo / Mistral (7B).
* **Techniques:** Model fine-tuning using the Hugging Face Trainer API, conversation modeling, and emotional tone design.

## Key Results and Findings
The fine-tuned model demonstrates a marked improvement in generating gentle, emotionally supportive responses compared to its base weights. The application successfully handles user inputs regarding stress and responds with contextual empathy, verified via a custom user interface.
