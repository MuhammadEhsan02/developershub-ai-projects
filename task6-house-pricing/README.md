# Task 6: House Price Prediction

## Task Objective
The objective is to build a predictive regression model that estimates real estate prices based on property features such as square footage, number of bedrooms, and location.

## Dataset Used
* **Name:** House Price Prediction Dataset
* **Source:** Kaggle

## Models and Techniques Applied
* **Data Preprocessing:** Feature scaling, selection, and handling of categorical location data.
* **Models:** Linear Regression and Gradient Boosting Regressor.
* **Evaluation Metrics:** Mean Absolute Error (MAE) and Root Mean Squared Error (RMSE).

## Key Results and Findings
The regression models were successfully trained and evaluated. Visualizations comparing predicted versus actual house prices indicate strong model performance. The MAE and RMSE metrics confirm the model's reliability in estimating property values based on the selected spatial and structural features.
