# Task 1: Exploring and Visualizing a Simple Dataset

## Task Objective
The objective of this task is to load, inspect, and visualize a standard dataset to understand data trends, feature distributions, and relationships between variables using exploratory data analysis (EDA) techniques.

## Dataset Used
* **Name:** Iris Dataset
* **Source:** Loaded via the Seaborn library.

## Models and Techniques Applied
* **Data Manipulation:** `pandas` (loading, `.head()`, `.info()`, `.describe()`).
* **Data Visualization:** `matplotlib` and `seaborn`.
* **Techniques:** Scatter plots for feature relationships, histograms for value distributions, and box plots for outlier detection.

## Key Results and Findings
The exploratory data analysis successfully highlighted the distinct clustering of Iris species based on sepal and petal measurements. The box plots effectively identified minor outliers in feature distributions, and the histograms provided a clear view of the normal distribution of the dataset's numerical attributes.
